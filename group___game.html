float get_delta ( )

Get delta (time between each frame).

This can be used as a scale for movement or changes over time so then each frame is consistent.

Ex. set_position(my_entity, get_position(my_entity) + (move*speed*get_delta()));`

void set_slot ( uint  pSlot)

Set current save slot.

This can be set once and it will stay the same for the rest of the game. This allows you to call save_game/open_game during the game without worrying about which slot to use (like checkpoint and such).
