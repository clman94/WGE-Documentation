Detailed Description

Basic entity reference.

Nothing special about it other than it keeps a reference to an entity stored in the engine.

Entities are basic graphical objects in the engine. They are typically very dynamic and should be used frequently as a building block. Entities come in different flavours including sprite(plain and simple), character, and text.

Since this is only a reference, assigning 2 entity objects together will result in 2 entity objects that point to the same entity. Which means they would be modifying the same entity.

Deleting this object will not delete the entity it references. You will need to use remove_entity.
