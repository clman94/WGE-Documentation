void player::focus ( bool  pIs_focus = true)

Set the focus of the camera to either focus on the player or freely move around.

When function like set_focus are used, the focus on the player is automatically removed.

This function can be called without any parameters to focus on player.

void set_tile ( const string &in  pAtlas,
vec  pPosition,
int  pLayer,
int  pRotation 
)

Set a tile in the tilemap.

Any overlapping tiles will be replaced. The tilemap is optimized for speed and not flexibility so it is recommended use entities over the tilemap any time you can.

Parameters
pAtlasThe atlas entry in the selected tilemap texture.
pPositionPosition of tile. Offsets(decimal) is allowed but it might be awkward to work with.
pLayerHigher value will result in the tile closer to the camera and farther if lower.
pRotationSimple value from 0-3?: 0 = 0 degrees; 1 = 90 degrees; 2 = 180 degrees; 3 = 270 degrees.
