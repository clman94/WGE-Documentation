Enumeration Type Documentation

enum anchor

Basic anchoring of a graphical entity.

Anchoring is the offsetting of a graphical object from the actual position based on its size. For example, if we choose "center", the object that is rendered is centered at the position.

See also
set_anchor
enum fixed_depth
Enumerator
